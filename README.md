# Supply Chain Intelligence Dashboard

Executive-grade supply chain analytics dashboard for Beauty & Personal Care.

## Features

* KPI tracking (Revenue, Profit, Margin, Risk)
* Scenario simulation (demand & cost shocks)
* Supplier performance analysis
* Logistics and route optimization insights
* SKU-level drill-down
* Interactive filtering
* AI-driven executive insights

## Tech Stack

* HTML, CSS, JavaScript
* Chart.js (CDN)

## Usage

Open `index.html` in any browser.

## Deployment

You can deploy instantly using GitHub Pages.
